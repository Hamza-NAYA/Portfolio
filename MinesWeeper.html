Mines Weeper

Code Link

Project Overview

This project aims to develop a Minesweeper game using object-oriented programming and recursion. The objective is to uncover and defuse a grid-based terrain littered with mines represented by a matrix of squares. Clicking on a square reveals its content.

The game includes the following scenarios:
 - Revealing a mine results in game loss
 - Revealing an empty square triggers a chain reaction to uncover adjacent empty squares
 - Revealing an empty square adjacent to mines displays a number indicating nearby mines (1 to 8)

Initially, the player makes a random click which determines mine placement, ensuring the first clicked square is never a mine. The classic Minesweeper features of flagging (indicating suspected mines) and marking with a question mark (uncertain status) are also implemented.

Difficulty levels allow customization of the game experience, influencing map size and mine count. The number of mines per map is dynamically generated within predefined minimum and maximum values, varying based on selected difficulty.

A timer is implemented to track game duration, adding a competitive element and motivating personal improvement, displayed within the game window. Additionally, a reset button is included to allow game restart.

This project combines strategic gameplay with interactive features and customization, offering a challenging and engaging Minesweeper experience.
