Project Overview

This project involves creating an incremental clicker game, where players progress through various stages:
 - Define a captivating universe/theme to ensure the game's appeal and coherence.
 - Clicking earns points for the user.
 - Accumulated points drive overall player progression, unlocking levels or other items.

The game includes:
 - A shop where points can be spent on items and bonuses.
 - Items enhance manual (clicking) and automatic (idle) point generation, while bonuses         improve these items.
 - Players can generate diverse resources by clicking on the game environment.
 - Costs for items and bonuses increase exponentially,
   enhancing point generation or improving existing items.
 - Persistent game progress is ensured via regular state storage in localStorage, preventing    loss of player advancement during interruptions.

This project offers an engaging experience by combining incremental gameplay mechanics with strategic resource management and persistent player progression.
